# Starter Boilerplate
Boilerplate for markup projects frontend with Pug, Nunjucks, Sass, Svg sprites, ES6, Eslint.

## Requirements
* Node >= 10.0.0
* Gulp cli >= 2.0.1

## Guide
#### Install dependencies
```commandline
npm install
```

#### Start local and hot-reloads
```commandline
npm start
```
```commandline
Local server http://localhost:8080/
```

#### Create build
```commandline
npm run build
```
